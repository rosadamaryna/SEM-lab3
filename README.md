# Knapsack Multi-Solver (Lab 3) 🎒

Інтерактивний додаток на Python для моделювання та порівняльного аналізу 5 методів розв'язання класичної **задачі про рюкзак (0/1 Knapsack Problem)**.

## 🚀 Про проект
Цей проєкт розроблено в межах лабораторної роботи №3 з курсу моделювання систем. Основна мета — продемонструвати роботу алгоритму динамічного програмування та порівняти його з альтернативними підходами.

### Реалізовані методи:
1. **Brute Force** (Груба сила)
2. **Recursive Approach** (Рекурсія)
3. **Dynamic Programming** (Динамічне програмування з візуалізацією матриці)
4. **Greedy Algorithm** (Жадібний вибір)
5. **Branch and Bound** (Метод гілок і меж)

## 🛠 Технології
- **Мова:** Python 3.10+
- **Інтерфейс:** Streamlit
- **Аналіз даних:** Pandas, NumPy
